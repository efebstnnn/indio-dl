# Inter font

Bu klasördeki `.ttf` dosyaları [Inter](https://rsms.me/inter/) font ailesinden türetilmiştir (Fontsource paketi üzerinden).

- **Regular** — arayüz metni
- **Medium / SemiBold** — vurgulu metin ve butonlar

Inter [SIL Open Font License 1.1](https://scripts.sil.org/OFL) altında dağıtılabilir.

Eksik dosyalar için: `python scripts/download_fonts.py`
