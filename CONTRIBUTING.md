# Katkı

Teşekkürler — katkılar memnuniyetle karşılanır.

## Hızlı başlangıç

```bash
git clone https://github.com/efebstnnn/yt-dlp-uygulama.git
cd yt-dlp-gui
pip install -r requirements.txt
python main.py
```

## Issue açmadan önce

- Python ve yt-dlp sürümlerini yaz
- Mümkünse hata mesajının tamamını ekle
- Aynı issue’nun açılıp açılmadığına bak

## Pull request

1. Küçük, odaklı değişiklikler tercih edilir
2. `python -m py_compile` ile dosyaların derlendiğinden emin ol
3. PR açıklamasında **ne** ve **neden** yaz

## Kod stili

- Mevcut dosyalardaki isimlendirme ve Türkçe arayüz metinlerine uy
- Gereksiz bağımlılık ekleme
